# Web Test Automation Framework (POM)

Framework kiểm thử tự động cho trang web sử dụng mô hình Page Object Model (POM).

## Cấu trúc thư mục
- `pages/`: Chứa các class đại diện cho từng trang web
- `tests/`: Chứa các test case
- `utils/`: Chứa các tiện ích hỗ trợ

## Thiết lập
1. Cài đặt các thư viện:
   ```bash
   pip install -r requirements.txt
   ```
2. Chạy kiểm thử:
   ```bash
   pytest
   ```
